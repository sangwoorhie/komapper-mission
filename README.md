# Mission

## Installation

```bash
$ npm install
$ cd backend > npm install
$ cd frontend > npm install
```

## Running the app

```bash
$ npm run start
```

## 🔎주요 기능

- 회원가입, 로그인, 로그아웃
- 회원정보 수정, 회원탈퇴, 유저목록조회, 유저E-mail 조회
- 비디오 생성(업로드), 비디오 목록조회, 비디오 ID조회, 비디오 다운로드
  <br>
  <br>

## 🖋️Tech Stack

- Typescript 5.3.3
- Node.js 18.17.0
- NestJS 10.3.2
- HTML, CSS, JavaScript
- Postgres, Sql Query Database
- Git, Github

의존성 설치
npm i
cd backend npm i
cd frontend npm i

서버 실행
루트 디렉토리에서 npm start 후 localhost:3001 실행

Front-end
사용 스택 : html, js, css

Back-end
사용 스택 : Nest.js

Database
사용 스택 : PostgreSQL

# Mission

## Installation

```bash
$ root directory > npm install
$ cd backend > npm install
$ cd frontend > npm install
```

## Running the app

```bash
$ root directory > npm run start
```

## 🔎주요 기능

- 회원가입, 회원 목록조회, 회원 ID로 찾기, 회원삭제, 회원정보 수정
- 로그 목록조회, 로그 ID로 찾기
  <br>
  <br>

- Typescript 5.3.3
